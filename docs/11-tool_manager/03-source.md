## Adding source

<p align="center">

| Action           | Description                                                                                    |
|------------------|------------------------------------------------------------------------------------------------|
| From browser     | Select files from browser.                                                                     |
| From CSV         | Add files from a CSV list                                                                      |
| From VUnit       | Add files from a [ VUnit ]( http://vunit.github.io/ ) run.py                                   |
</p>

The accepted CSV format is as follows:

```
file_0.sv
file_1.vhd
file_2.vhd
```

or

```
file_1.vhd, lib_0
file_2.vhd
file_3.vhd, lib_0
```
